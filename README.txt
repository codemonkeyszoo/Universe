README for Universe

What it is
==========

Universe is a small program to demonstrate the motion of material points under the force of gravity.


How to use
==========

1) Download latest version.

2) Run the program:

     java -jar Main.jar

   main window should appear on screen :)

3) Add some points by clicking mouse.

4) Click "Run" button.

   Results should appear in program window.

   To stop points press "Stop" button.
